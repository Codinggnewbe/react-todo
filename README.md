# Todo List Project

React와 Express를 사용하여 구현한 할 일 관리(Todo List) 웹 애플리케이션입니다.  
기본적인 CRUD 기능부터 체크박스 상태 관리, 세부 내용 관리까지 구현하는 것을 목표로 했습니다.

---

## 주요 기능

- 할 일 CRUD (작성 / 조회 / 수정 / 삭제)
- 체크박스를 통한 완료 여부 관리
- 할 일별 세부 내용(specifics) 추가 및 삭제
- React Router를 이용한 페이지 이동

---

## 사용 기술

### Frontend
- React
- styled-components
- axios

### Backend
- Express

## 구현 과정에서 해결한 문제

### 1. 체크박스와 텍스트가 동시에 풀리는 문제

체크박스를 클릭했을 때 텍스트와 상태가 함께 변경되지 않는 문제가 발생했습니다.

- 서버에 `app.patch` 엔드포인트를 추가하여 완료 여부를 서버 상태로 관리하도록 수정했습니다.
- 클라이언트(MainPage)에서도 `axios.patch`를 사용해 서버와 상태를 동기화했습니다.

---

### 2. 체크박스와 텍스트 상태가 연동되지 않는 문제

- 체크박스에 `checked={todo.done}` 속성을 추가하여 상태를 명확히 연결했습니다.
- 개발자 도구(F12)에서 오류를 확인한 결과, 일부 todo 객체에 `done` 값이 없다는 것을 확인했습니다.
- 서버에서 `done` 값이 `undefined`일 경우 `false`를 기본값으로 설정하여 문제를 해결했습니다.

---

### 3. 체크박스 UI 커스터마이징

- 기본 체크박스 UI를 버튼과 유사하게 맞추기 위해 `CheckBox.jsx` 컴포넌트를 별도로 구현했습니다.
- 이를 통해 체크박스 크기와 클릭 영역을 보다 명확하게 제어할 수 있었습니다.

---

### 4. 삭제 기능 구현 중 경로 및 배열 처리 문제

- 삭제 시 기존 배열을 직접 수정하지 않고, `filter`를 사용해 새로운 배열을 생성하여 상태를 갱신했습니다.
- 체크된 일정 삭제 기능은 `done === false`인 항목만 남겨 새로운 배열로 대체하는 방식으로 구현했습니다.
- 삭제 후 클라이언트 상태 불일치를 방지하기 위해, 삭제 요청 이후 `GET` 요청으로 최신 데이터를 다시 불러오도록 처리했습니다.
- 삭제 기능에서 같은 URL에 여러 기능이 겹쳐 혼동 가능성이 있어,  
  체크된 일정 삭제는 `/todos/done`으로 URL을 분리했습니다.

---

### 5. 개별 할 일 삭제 기능

- 특정 할 일을 삭제하기 위해 URL에 `todoId`를 포함해야 했고, 이를 위해 문자열 결합(`""`)이 아닌 템플릿 리터럴(`\``)을 사용했습니다.
- `axios.delete(`${url}/${todoId}`)` 형태로 요청을 보냈습니다.
- 서버에서는 전체 todos 중 전달받은 id와 일치하는 항목을 제외한 나머지를 다시 배열로 구성하여 삭제를 처리했습니다.

---

### 6. 할 일 수정 기능 구현

- 수정 페이지 진입 시 `GET` 요청을 통해 기존 `title`과 `content`를 불러왔습니다.
- 두 개의 state를 동시에 갱신해야 하므로 `then` 블록 내부에서 각각 `setTitle`, `setContent`를 처리했습니다.
- 수정 기능에서는 `id`, `specifics` 등을 변경할 필요가 없기 때문에 `title`과 `content`만 서버로 전송하도록 정리했습니다.
- 서버에서는 기존 todo 객체에 수정된 값만 반영하도록 처리했습니다.

---

### 7. 세부 내용(specifics) 개별 삭제 기능

- Todo 삭제는 `MainPage`에서 관리하지만, specifics 삭제는 `DetailListPage`에서 관리하도록 역할을 분리했습니다.
- 초기 구현 과정에서 서버에서 500 에러가 발생했는데, 이는 `comments`라는 잘못된 필드를 사용했기 때문이었습니다.
- 실제 데이터 구조에 맞게 `todo.specifics`로 수정하여 정상 동작하도록 해결했습니다.

---

## 해결하지 못한 문제

- 체크박스와 휴지통 아이콘 버튼 간의 세로 정렬이 완벽하게 맞지 않는 문제
- `flex` 정렬 및 스타일 조정을 시도했으나, 체크박스와 이미지 요소 간의 기본 렌더링 차이로 인해 완전히 해결하지는 못했습니다.
